Wiring Connections

Connect your components to the ESP32 exactly as shown below. Both ADXL345 and OLED share the same I2C bus on GPIO 21 & 22.

Component Pin/Wire ESP32 Pin
ADXL345 VCC 3.3V

Buzzer Negative (-) GND

Quick Setup Guide

‚úÖ 5 Simple Steps

Step 1: Install Arduino Libraries

  • Open Arduino IDE ‚Üí Tools ‚Üí Manage Libraries
  • Search & Install: Adafruit ADXL345
  • Search & Install: Adafruit SSD1306
  • Search & Install: Adafruit GFX Library
  • Search & Install: Blynk by Blynk

Step 2: Wire Components

  • Follow the wiring table above perfectly
  • ADXL345 & OLED share I2C bus (GPIO 21 = SDA, GPIO 22 = SCL)
  • Buzzer on GPIO 2, LED on GPIO 4
  • All GND pins connected together

Step 3: Update WiFi Credentials

  • Replace YOUR_SSID with your WiFi name
  • Replace YOUR_PASSWORD with WiFi password
  • Keep the Blynk token as is (already configured)

Step 4: Upload Code

  • Select Board: Tools ‚Üí Board ‚Üí ESP32 ‚Üí ESP32 Dev Module
  • Select Port: Tools ‚Üí Port ‚Üí Choose your COM port
  • Click Upload button (‚Üí arrow icon)
  • Wait for "Upload completed" message

Step 5: Test & Monitor

  • Watch OLED display for real-time data
  • Open Blynk app to see live updates
  • Gently shake the device to trigger earthquake detection
  • Buzzer & LED should activate when threshold exceeded

Adjustable Settings

‚öôÔ∏è Fine-tune Detection

THRESHOLD (15.0): Acceleration level to trigger alarm in m/s¬≤. Higher = less sensitive, Lower = more sensitive. Try 10-20.

ALARM_DURATION (5000): How long buzzer/LED stays on after detection in milliseconds. 5000 = 5 seconds.

BUZZ_PATTERN (100): Buzzer on/off interval in milliseconds. Lower = faster blinking/buzzing.

Troubleshooting

‚ùå ADXL345 Not Detected

Check GPIO 21/22 connections for SDA/SCL. Verify I2C address 0x53. Try I2C scanner from Arduino examples.

‚ùå OLED Not Displaying

Verify 3.3V power. Check I2C address 0x3C. Both sensors must share GPIO 21 & 22.

‚ùå Blynk Not Connecting

Verify WiFi SSID/password. Check Serial Monitor for WiFi connection status. Ensure Blynk device is added in app.

‚ùå Buzzer/LED Not Working

Test with multimeter. Check GPIO 2/4 output. Verify buzzer polarity. Add 220Œ© resistor for LED.
